# computorv2

## Some improvements over the subject

- Evaluation of the expression can be done without using the `= ?` syntax

## Bonus

- Support for constants:
	- `e`
	- `inf`
	- `nan`
	- `pi`
	- `tau`
- Support for built-in functions:
	- `abs`
	- `cos`
	- `exp`
	- `log`
	- `sin`
	- `sqrt`
	- `tan`
- Functions with multiple arguments
- Built-in commands
- Matrix inversion
- Commands history
